 ## Image generation rules

When generating decorative design assets (icons, ornaments, patterns, 
textures, wedding elements, luxury graphics):
- Always use fal model: fal-ai/flux-pro/v1.1
- Always add: isolated on white background, no text, ultra detailed
- Save to /public/assets/

When drafting or iterating on image concepts quickly:
- Use fal model: fal-ai/seedream-v4 (or fal-ai/fast-sdxl)
- Save drafts to /tmp/drafts/

When generating UI screenshots or mockup images:
- Use fal model: fal-ai/nano-banana-pro
